# LPs que desenvolvi no Cruzeiro-SAF

*Todos os direitos guardados e reservados para uso do clube Cruzeiro Esporte Clube - SAF*

**Desenvolvido por:** Marco Túlio Paiva Repoles

## Sobre os Projetos

Coleção completa de landing pages desenvolvidas para o Cruzeiro Esporte Clube - SAF, focando em experiência do usuário, responsividade e identidade visual do clube.

## Páginas Desenvolvidas

### Transparência e Governança

- **Atas de Assembleia** - AGE, AGO, Conselho de Administração e Fiscal
- **Demonstrativos Financeiros** - Relatórios anuais 2022-2024
- **Portal de Transparência** - Interface completa para documentos obrigatórios

### Experiência do Torcedor

- **Guia do Mineirão** - Informações completas sobre o estádio
- **Mascotes Oficiais** - Raposão, Raposinho e Cabulosa com galerias interativas
- **Uniformes 2025** - Showcase das camisas com detalhes técnicos
- **Lojas Oficiais** - Mapa e informações de todas as 23 unidades

### Estrutura do Clube

- **Toca da Raposa 2** - Tour virtual do centro de treinamento
- **Patrocinadores** - Apresentação de parceiros e masters

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Flexbox/Grid
- JavaScript vanilla
- Design responsivo mobile-first
- Otimização de performance
- Acessibilidade web

## Características Técnicas

- **Responsivo** - Adaptação perfeita para todos os dispositivos
- **Performance** - Carregamento otimizado e experiência fluida
- **Acessibilidade** - Seguindo padrões WCAG
- **SEO** - Estrutura otimizada para mecanismos de busca
- **Identidade Visual** - Fiel às cores e tipografia do Cruzeiro
